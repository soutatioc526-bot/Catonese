# 粤语

零构建、纯静态版本，可直接部署到 GitHub Pages。

## 使用

- 本地：双击 `index.html`。
- GitHub Pages：把本目录全部文件推送到仓库，在 `Settings → Pages` 选择从分支部署。
- 无需 npm、Node.js、构建命令或环境变量。

## 学习规则

每句第一次播放为逐字听音，第二次为快速连读。七个阶段各含 20 个词句；完成当前阶段的 20 个词句，并在 6 题随机测试中取得至少 75 分，才会解锁下一阶段。

自由练习不计分，答错可立即重答；阶段测试计分，题型会在听力、粤式文字辨义、普通话选粤语之间随机切换。

## 音频说明

项目内置香港母语者录制的带调粤语音节。逐字播放保留音节间隔；快速连读采用相同母语音节的快速无间隔串联。粤拼音节按钮可单独点读。当前版本没有把浏览器合成语音冒充为母语者整句录音。

## 问句

“AI”页在静态部署中使用课程内置数据回答，不上传内容，也不需要密钥。公开 GitHub Pages 不应写入任何 API 密钥；需要云端模型时，应另接私有的 Serverless API 代理。
